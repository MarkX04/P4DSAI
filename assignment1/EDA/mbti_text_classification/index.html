🔧 Feature Engineering

Three numerical features are derived from the raw posts column, plus four binary trait columns from the MBTI type string.

Derived Features

  • word_count – number of whitespace-separated tokens
  • char_count – total character length
  • num_posts – count of |||-separated posts

Trait Columns

  • I_E – Introversion / Extraversion (1st letter)
  • N_S – Intuition / Sensing (2nd letter)
  • T_F – Thinking / Feeling (3rd letter)
  • J_P – Judging / Perceiving (4th letter) 
df["posts"] = df["posts"].astype(str)

🧹 Text Cleaning

Pipeline: lowercase → remove URLs → replace ||| → strip non-alpha → collapse whitespace → remove stopwords (words ≤ 2 chars filtered).

Steps

  • Lowercase all text
  • Remove URLs (http\S+)
  • Replace post separators ||| with space
  • Strip non-alphabetic characters
  • Collapse multiple whitespace
  • Remove NLTK English stopwords & words ≤ 2 chars

Result Columns

  • clean – cleaned text (stopwords kept)
  • clean_ns – cleaned text without stopwords 
STOP_WORDS = set(stopwords.words("english"))

def clean_text(text):
    text = text.lower()
    text = re.sub(r"http\S+", " ", text)
    text = text.replace("|||", " ")
    text = re.sub(r"[^a-z\s]", " ", text)
    text = re.sub(r"\s+", " ", text)
    return text

def remove_stopwords(text):
    return " ".join(
        [w for w in text.split() if w not in STOP_WORDS and len(w) > 2]
    )

df["clean"]    = df["posts"].apply(clean_text)
df["clean_ns"] = df["clean"].apply(remove_stopwords)

def vocab_stats(texts):
    tokens = " ".join(texts).split()
    return len(set(tokens)) / len(tokens)

for t, g in df.groupby("type"):
