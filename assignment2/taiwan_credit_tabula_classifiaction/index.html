Conclusion — Final Rankings

All metrics from the held-out test set. Ranked by F1-score on the Default (positive) class.

Rank Model Test Acc Precision Recall F1 ↓ AUC-ROC Train Acc Val Acc Overfit
Random Forest achieves the highest F1 (0.4645). Gradient Boosting leads on AUC-ROC (0.7824) — best at ranking defaulters. Neural Network (MLP) has the highest recall (41.47%), detecting the most actual defaults. Logistic Regression has the highest precision (70.66%) but misses most actual defaults (recall 23.69%).
⚠️ Class imbalance (≈78% No Default / 22% Default): all models show low recall on the Default class (24–41%). For production credit scoring, consider threshold tuning, SMOTE oversampling, or cost-sensitive learning to improve Default detection.
