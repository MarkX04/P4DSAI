📋 Assignment Overview

This assignment demonstrates the application of machine learning classification techniques across different data modalities, following the exploratory data analysis from Assignment 1.

  • Text Classification: TF-IDF + Traditional ML (LR, NB, SVM, RF, MLP, XGBoost)
  • Tabular Classification: Feature engineering + Ensemble methods
  • Image Classification: Transfer learning with pretrained CNNs
  • Evaluation: Comprehensive metrics, confusion matrices, and model comparison
