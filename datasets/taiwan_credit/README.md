# Taiwan Credit Card Default Dataset

**Source:** [UCI Machine Learning Repository — Dataset #350](https://archive.ics.uci.edu/dataset/350/default+of+credit+card+clients)

## Overview

| Attribute      | Detail                                                     |
| -------------- | ---------------------------------------------------------- |
| Instances      | 30,000 credit card clients                                 |
| Features       | 23 input features + 1 binary target                        |
| Domain         | Finance / Credit Risk Modelling                            |
| Target         | `Default_Payment_Next_Month` — 1 = default, 0 = no default |
| Period         | April 2005 – September 2005, Taiwan                        |
| Missing Values | None                                                       |

## Feature Dictionary

| Original | Renamed                    | Type        | Description                                                                                      |
| -------- | -------------------------- | ----------- | ------------------------------------------------------------------------------------------------ |
| X1       | Credit_Limit               | Numeric     | Amount of given credit (NT$)                                                                     |
| X2       | Gender                     | Categorical | 1 = Male, 2 = Female                                                                             |
| X3       | Education                  | Categorical | 1 = Graduate School, 2 = University, 3 = High School, 4 = Others                                 |
| X4       | Marriage                   | Categorical | 1 = Married, 2 = Single, 3 = Others                                                              |
| X5       | Age                        | Numeric     | Age in years                                                                                     |
| X6–X11   | PAY_1 – PAY_6              | Ordinal     | Repayment status Sep–Apr 2005 (-2=no consumption, -1=paid duly, 0=revolving, 1–8=months delayed) |
| X12–X17  | BILL_AMT_1 – 6             | Numeric     | Bill statement amount Sep–Apr 2005 (NT$)                                                         |
| X18–X23  | PAY_AMT_1 – 6              | Numeric     | Previous payment amount Sep–Apr 2005 (NT$)                                                       |
| Y        | Default_Payment_Next_Month | Binary      | 1 = default, 0 = no default                                                                      |

## Data Cleaning Applied

| Step | Action                                                |
| ---- | ----------------------------------------------------- |
| 1    | Rename columns from `X1-X23 / Y` to descriptive names |
| 2    | `Education` codes 0, 5, 6 → 4 (Others)                |
| 3    | `Marriage` code 0 → 3 (Others)                        |
